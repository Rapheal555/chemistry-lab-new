## Adjusting Equipment Settings âš™ï¸

Want to change how fast it filters? Adjust these:

### Filter Pore Size

- **Smaller** = Catches finer particles but SLOWER
- **Larger** = Faster but may miss small particles
- Try: 5-50 Âµm

### Vacuum Pressure

- **No vacuum (0)** = Gravity only, SLOW
- **High vacuum (500-1000)** = Much FASTER
- Try: 300 mbar to start

### Other Factors:

- **Viscosity**: Set by which liquid you choose
- **Filter Area**: Larger = faster (like a bigger funnel)

## Try These Experiments

### Experiment 1: Speed Test

- Use Sand + Water
